# OpenDot

OpenDot is a minimal blogging platform built to explore the **fundamentals of how the web actually works**.

The goal of this project is not to build a production-ready application, but to understand the **core mechanics behind the web stack** — including how browsers communicate with servers, how HTTP requests are handled, how routing works, and how web applications were built before modern frameworks introduced heavy abstractions.

OpenDot allows users to share short blog posts about things they are curious about.
